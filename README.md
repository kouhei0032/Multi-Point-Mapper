# プロジェクト名：Multi-Point-Mapper

お客様名と住所を入力することで、複数地点を地図にマッピングするシステムです。経路の算出をすることも可能になっています。

## 特徴：Features

* **機能1**: 複数のお客様所在地を地図上にマッピングできます。
* **機能2**: マッピングした住所の効率的な訪問経路を算出できます。
